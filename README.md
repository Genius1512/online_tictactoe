# Online Tictactoe

## How to use

### Clone the project

```bash
git clone github.com/genius1512/online_tictactoe
```

### Move to the online_tictactoe/scripts direcotry

```bash
cd ./online_tictactoe/scripts
```

### Start a server

```bash
python main.py
server
<port>
```

### Connect two clients

```bash
python main.py
client
<ip>
<port>
```

2x

### Start playing
